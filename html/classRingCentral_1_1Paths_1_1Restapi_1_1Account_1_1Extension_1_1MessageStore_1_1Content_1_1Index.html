Returns media content of a message attachment. The content is typically an audio file (audio/mpeg or audio/wav) for voicemails, TIFF or PDF for faxes and image/audio/video for MMS. This API must be called via media API entry point, e.g. https://media.ringcentral.com HTTP Method: get Endpoint: /restapi/{apiVersion}/account/{accountId}/extension/{extensionId}/message-store/{messageId}/content/{attachmentId} Rate Limit Group: Medium App Permission: ReadMessages
