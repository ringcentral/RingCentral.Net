◆ prompt

string RingCentral.AuthorizeRequest.prompt
getset

Specifies which login form will be displayed. Space-separated set of the following values:

  • login - RingCentral native login form,
  • sso - Single Sign-On login form,
  • consent - form to show the requested scope and prompt user for consent. Either login or sso (or both) must be specified. The default value is login sso Default: login sso
